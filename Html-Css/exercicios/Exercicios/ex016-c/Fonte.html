Fontes

Medidias

Medidas absolutas:

  • cm - centimetros
  • mm - milimetros
  • in - polegada
  • *px - pixel (16 px é o tamanho normal)
  • pt - ponto (não recomendado)
  • pc - pyca (não recomendado)

Medidas relativas:
  • *em - relativo ao tamanho m minusculo fonte
  • ex - relativo a altura x da fonte atual
  • rem - relativo a fonte do body
  • vw - relativo a view port em % - tamanho de tela
  • vh - a altura da view port

1em=16px

Orienta-se utilizar Pixels ou EM para font-size, porém o melhor é o EM.

Peso e estilo

Fonte work sans utilizada para demonstrar o peso da fonte.
Weigth = peso Font-weigth = ligthter, normal, bold, bolder Ou de 100 a 900.

Font-style = para escolher italico ou normal. A fonte escolhida deve conter opções tanto em negrito quanto em italico.

Sublinhado = text-decoration=underline

Width = Largura
Heigth = Altura

Condensação

  • Font-family
  • Font-weight
  • Font-size
  • Font-style
*Shorthand= font.
EX: font: italic bolder 3m 'work sans, sans-serif';
A ordem é a seguinte: font-style; font-weigth; font-size; font-family.

Google fonts

Colocando fontes utilizando o google fonts clicando em Embed - /@import e colocando na primeira linha do style e depois utilizando normalmente.

Fontes externas

Fontes baixadas de sites como "dafont.com". Abresse então o arquivo ZIP e o conteudo deve ser adicionadao aqui na pasta "ex014" por exemplo, como se faz com uma imagem. Com isso como "regra"(logo deve ser inserido logo abaixo do "style") com o "@font-face: NOME ATRIBUIDO A NOVA FONTE// SCR: URL ('aqui o nome do arquivo'). Depois pode ser utilizado como qualquer outra.

Para identificar uma fonte de algum site pode-se utilizar a extenção no goolge chamda "fonts ninja"
